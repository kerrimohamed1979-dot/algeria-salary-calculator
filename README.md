# Calculateur de Salaire Algérie (2026)

Ce projet est un calculateur de salaire simple pour le système fiscal algérien (IRG 2022/2026).

## Comment l'utiliser
1. Ouvrez le dossier `C:\Users\User\Documents\algeria-salary-calculator`.
2. Double-cliquez sur le fichier `index.html`.
3. Le calculateur s'ouvrira dans votre navigateur web.

## Fonctionnalités
- Calcul de la Retenue SS (9%)
- Calcul du Salaire Imposable
- Calcul de l'IRG (Barème progressif + Abattements 40% et complémentaires)
- Estimation des Allocations Familiales (300 DA/enfant + 800 DA Salaire Unique)

## Structure des fichiers
- `index.html`: La structure de la page.
- `style.css`: Le style (couleurs, mise en page).
- `script.js`: La logique de calcul (Javascript).
